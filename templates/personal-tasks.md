# Personal Tasks — Your Custom Recurring Tasks

Add your own recurring tasks here. These run after the default tasks (see
[default-tasks](default-tasks.md)) and before inbox or ActingWeb tasks.

This file is yours — it will never be overwritten by upgrades. The default-tasks.md
file may be updated with new features, but your personal tasks here are preserved.

Follow the same format as default-tasks.md:
- Give each task a numbered section heading
- Specify the **Type** and any **MCP required**
- Add **Frequency** constraints if it shouldn't run every cycle
- Keep tasks small — each should complete in under 2 minutes
