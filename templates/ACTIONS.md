# Action Items

This file is your dashboard. The agent updates it every run. You act on it when you can.

## How it works

**For you**: Review items below and act on them. When done, ~~strikethrough~~ the item (or use the checkbox in Obsidian) — the agent will remove it on the next run and note it in the log.

**To give the agent work**: Drop a `.md` or `.txt` file in [`INBOX/`](INBOX/) with instructions, or use the inline comment line (`>`) below any action item to give quick instructions — the agent reads and processes these each run. For iterative tasks (e.g. revise a draft, redo research), inline comments often replace the need for INBOX files.

**Email drafts**: Drafts live in [`output/emails/`](output/emails/) with a `status:` frontmatter. Change `status: pending` to `status: approved` and the agent sends it next run. Or set `status: discard` to archive it.
